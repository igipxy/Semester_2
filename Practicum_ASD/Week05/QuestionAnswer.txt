int minIndex = i;
for (int j = i + 1; j < n; j++) {
    if (data[j] < data[minIndex]) {
        minIndex = j; // Update minIndex if a smaller value is found
    }
}

3. The condition while (j >= 0 && data[j] > key) is for

    1. j >= 0 (Boundary Check): This prevents an ArrayIndexOutOfBoundsException. It ensures the loop stops once it has compared 
    the key against all elements to the left (reaching the start of the array).

    2. data[j] > key (Comparison): This determines if the current element is larger than the "key" being inserted. 
    If it is, the loop continues so that the larger element can be shifted to the right, making space for the key to eventually be placed in its correct sorted position.

4. In the context of Insertion Sort, this specific line is responsible for shifting.

    1. Function: It copies the value of the element at index j to the position directly to its right (j + 1).

    2. Why it's done: Because the algorithm has found that data[j] is larger than the key, it moves data[j] one spot to the right to create a "vacancy" or gap. This process repeats until the correct spot for the key is found.

import java.util.Scanner;

public void inputStudentData(Student[] listStudents, int idx) {
    Scanner sc = new Scanner(System.in);
    
    for (int i = 0; i < idx; i++) {
        System.out.println("Enter data for student " + (i + 1));
        
        System.out.print("NIM         : ");
        int nim = sc.nextInt();
        sc.nextLine(); 
        
        System.out.print("Name        : ");
        String name = sc.nextLine();
        
        System.out.print("Class       : ");
        String studentClass = sc.nextLine();
        
        System.out.print("GPA (IPK)   : ");
        double gpa = sc.nextDouble();
        
        listStudents[i] = new Student(nim, name, studentClass, gpa);
        System.out.println("-------------------------");
    }
}
